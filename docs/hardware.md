## Connections

| Pin  | Connection             |
| ---- | ---------------------- |
| A9   | Debug TX1              |
| A10  | Debug RX1              |
| PB12 | Button1     		        |
| PB13 | Button2     		        |
| PA7  | Button3     		        |
| PA6  | Button4 (Enc)          |
| PA0  | Enc1                   |
| PA1  | Enc2                   |
| PA2  | MIDI TX1 (Hex Buffer)  |

### Debugging with UART Adapter

Only connect TX to A9, RX to A10 and GND, no need for 3.3V connection
